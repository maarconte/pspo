# ⚠️ IMPORTANT: Variables d'environnement à mettre à jour

Avant de lancer l'application, vous devez mettre à jour votre fichier `.env.local` pour utiliser le préfixe `VITE_` au lieu de `REACT_APP_`.

## Changements requis dans `.env.local`

Renommez toutes vos variables d'environnement comme suit :

```bash
# AVANT (Create React App)
REACT_APP_FIREBASE_API_KEY=...
REACT_APP_FIREBASE_AUTH_DOMAIN=...
REACT_APP_FIREBASE_PROJECT_ID=...
REACT_APP_FIREBASE_STORAGE_BUCKET=...
REACT_APP_FIREBASE_MESSAGING_SENDER_ID=...
REACT_APP_FIREBASE_APP_ID=...

# APRÈS (Vite)
VITE_FIREBASE_API_KEY=...
VITE_FIREBASE_AUTH_DOMAIN=...
VITE_FIREBASE_PROJECT_ID=...
VITE_FIREBASE_STORAGE_BUCKET=...
VITE_FIREBASE_MESSAGING_SENDER_ID=...
VITE_FIREBASE_APP_ID=...
```

## Commandes mises à jour

- **Développement** : `npm run dev` (au lieu de `npm start`)
- **Build** : `npm run build`
- **Preview** : `npm run preview`
- **Deploy** : `npm run deploy` (inchangé)

## Dossier de build

Le dossier de build est maintenant `dist/` au lieu de `build/`.
